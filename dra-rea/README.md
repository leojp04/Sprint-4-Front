# IMREA Digital — Sprint 4 (Front-end Design Engineering)

Aplicação SPA desenvolvida para o desafio IMREA Digital, com foco em acessibilidade, responsividade e integração a serviços de agendamento. O projeto usa React 19 com Vite e TypeScript, Tailwind CSS para estilização utilitária e json-server para simular os endpoints da API Java hospedada remotamente.

## Tecnologias
- React 19
- Vite
- TypeScript
- Tailwind CSS 4
- React Router DOM
- React Hook Form + Zod
- ESLint
- json-server (mock da API)


## Estrutura de pastas
```
src/
  assets/
  components/
  data/
  hooks/
  pages/
  services/
  types/
  App.tsx
  main.tsx
  routes.ts
  index.css
```

## Como rodar localmente
1. Faça uma cópia de `.env.example` para `.env` e ajuste `VITE_API_URL`.
2. Instale as dependências:
   ```
   npm install
   ```
3. Inicie o ambiente de desenvolvimento:
   ```
   npm run dev
   ```
   A aplicação abrirá em `http://localhost:5173`.
